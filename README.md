# Tourism & Hotel Booking System (MySQL)

## Project Overview
The Tourism & Hotel Booking System is a relational database project built using MySQL.  
This project manages customer information, hotel details, room availability, bookings, and payments. It simulates a real-world hotel booking environment and generates business insights using SQL.

This project is designed for a **Data Analyst portfolio** to demonstrate SQL, database design, and data analysis skills.

---

## Business Problem
Tourism companies and hotel chains need a centralized system to:
- Manage customer bookings
- Track hotel and room availability
- Monitor revenue performance
- Analyze customer behavior
- Identify top-performing hotels and locations

This database helps answer key business questions using SQL.

---

## Database Structure

### Tables Included
- Customers – Stores customer details
- Hotels – Hotel information and location
- Rooms – Room types and pricing
- Bookings – Customer booking records
- Payments – Payment transaction details

### Relationships
- One Customer → Many Bookings  
- One Hotel → Many Rooms  
- One Room → Many Bookings  
- One Booking → One Payment
